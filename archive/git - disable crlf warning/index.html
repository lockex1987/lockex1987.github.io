Tắt cảnh báo CRLF

Khi commit git, chúng ta hay gặp cảnh báo sau:

warning: LF will be replaced by CRLF in ...
The file will have its original line endings in your working directory.

Ở Windows, ký tự xuống dòng là CRLF.

Ở Linux, Mac, ký tự xuống dòng là LF.

Có thể khó chịu nếu project gồm nhiều dev, có dev dùng Windows, có dev dùng Linux.

Git có thể xử lý việc này bằng cách tự động chuyển ký tự xuống dòng CRLF thành LF khi bạn thêm một file vào index, và ngược lại khi file đó được checkout về filesystem của bạn. Bạn có thể bật tính năng này với cấu hình core.autocrlf. Nếu bạn ở một máy Windows, thiết lập cấu hình này thành true. Cái này sẽ tự động chuyển LF thành CRLF khi check out.

git config --global core.autocrlf true

Nếu bạn là một lập trình viên Windows và làm một dự án chỉ cho Windows thì bạn có thể tắt tính năng này.

git config --global core.autocrlf false
