Sử dụng proxy trong git

Giả sử mạng của bạn chặn GitHub. Làm thế nào bạn có thể làm việc với git thông qua một proxy?

Sử dụng lệnh sau:

git config --global http.proxy http://proxyuser:proxypwd@proxy.server.com:8080

Ví dụ:

git config --global http.proxy http://192.168.103.31:80

Bỏ không sử dụng proxy nữa:

git config --global --unset http.proxy

Lấy thông tin proxy hiện tại:

git config --global --get http.proxy
git config --list

Bạn có thể thiết lập proxy riêng cho một URL cụ thể.
