Các phiên bản Linux Mint

Hiện tại, tôi đang sử dụng Linux Mint. Linux Mint là một bản phân phối dựa trên Ubuntu. Ubuntu thì nổi tiếng hơn Linux Mint.

Khi cài một số phần mềm, nó thường có các phiên bản khác nhau dựa trên Ubuntu 16.04 hay là Ubuntu 18.04. Do đó, tôi cần biết phiên bản Linux Mint của tôi dựa trên Ubuntu nào.

Đọc file /etc/os-release để lấy mã Ubuntu cũng như thông tin Linux Mint.

cat /etc/os-release

Kết quả:

NAME="Linux Mint"
VERSION="18.3 (Sylvia)"
ID=linuxmint
ID_LIKE=ubuntu
PRETTY_NAME="Linux Mint 18.3"
VERSION_ID="18.3"
HOME_URL="http://www.linuxmint.com/"
SUPPORT_URL="http://forums.linuxmint.com/"
BUG_REPORT_URL="http://bugs.launchpad.net/linuxmint/"
VERSION_CODENAME=sylvia
UBUNTU_CODENAME=xenial

Đọc file /etc/upstream-release/lsb-release để chỉ lấy thông tin Ubuntu.

cat /etc/upstream-release/lsb-release

Kết quả:

DISTRIB_ID=Ubuntu
DISTRIB_RELEASE=16.04
DISTRIB_CODENAME=xenial
DISTRIB_DESCRIPTION="Ubuntu 16.04 LTS"
