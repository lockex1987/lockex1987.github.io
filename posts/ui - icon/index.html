Sử dụng icon trên web

Icon đẹp, làm trang web màu sắc rực rỡ.

Dùng HTML entity thì có thể các font chữ khác nhau thể hiện khác nhau.

Dùng Font Awesome có thể đổi màu dễ dàng (color, background-color).

Tuy nhiên nhiều icon xấu, hoặc phải dùng bản Pro có phí.

Dùng Line Awesome.

icons8.com/line-awesome

Dùng tool Ico Moon để chỉ chọn ra các icon cần thiết.

icomoon.io
