AJAX

AJAX là viết tắt của Asynchronous JavaScript And XML. AJAX không phải là một ngôn ngữ lập trình hay một thư viện. AJAX cho phép bạn trao đổi dữ liệu với server ở background, cập nhật lại các phần của trang mà không phải load lại toàn bộ.

Để sử dụng AJAX chỉ bằng JS thuần, chúng ta sử dụng đối tượng XMLHttpRequest hoặc Fetch API. Chúng ta cũng có thể sử dụng các thư viện như jQuery, axios.

Demo

Dù bạn sử dụng cái gì thì cũng cần chú ý đến những cái sau:

  • encode tham số
  • Các dạng gửi dữ liệu khác nhau:
    • GET
    • POST form
    • POST JSON
    • POST upload
  • Xử lý dữ liệu trả về dạng HTML, JSON, binary
  • Hỗ trợ hiển thị upload progress
  • Hỗ trợ intercept:
    • Thêm CSRF token
    • Kiểm tra lỗi session timeout, xử lý lỗi Laravel validation
    • Handle lỗi nói chung
  • Xử lý nhiều request đồng thời và chờ tất cả request thành công
