Cài đặt và sử dụng WINE

WINE giúp chạy các ứng dụng Windows trên Linux. Nhiều ứng dụng chỉ có trên Windows như Photoshop, MS Office, nhiều game.

Cài đặt

Chúng ta sẽ cài đặt WINE trên Ubunut 20.10.

Phiên bản WINE trong kho Ubuntu Software là 5.0-3ubuntu1.

Để cài đặt phiên bản mới nhất (5.22), chúng ta làm các bước sau.

Cấu hình architecture của hệ thống

Thông thường Ubuntu 20.10 được cài với hệ thống kiến trúc 64-bit. WINE cần hỗ trợ đồng thời cả ứng dụng 32-bit. Chúng ta thêm kiến trúc 32-bit như sau:

sudo dpkg --add-architecture i386

Thiết lập PPA

Download key của WINE repository:

wget -nc https://dl.winehq.org/wine-builds/winehq.key

Thêm key:

sudo apt-key add winehq.key

Gộp 2 lệnh trên:

wget -qO - https://dl.winehq.org/wine-builds/winehq.key | sudo apt-key add -

Tiếp theo, chúng ta sẽ thêm repository của WINE.

Với Ubuntu 20.04:

sudo add-apt-repository 'deb https://dl.winehq.org/wine-builds/ubuntu/ focal main'

Với Ubuntu 20.10:

sudo add-apt-repository 'deb https://dl.winehq.org/wine-builds/ubuntu/ grovy main'

Cài đặt package

Cập nhật các package:

sudo apt update

Tiếp theo, chúng ta có thể cài đặt một trong 3 phiên bản (stable, development, staging):

Bản stable:

sudo apt install --install-recommends winehq-stable

Bản development:

sudo apt install --install-recommends winehq-devel

Bản staging:

sudo apt install --install-recommends winehq-staging

Xác nhận lại việc cài đặt:

wine --version




Cấu hình

Thực hiện lệnh sau:

winecfg

Nhấn nút "Install" để download và cài đặt WINE.

Trong quá trình cài đặt, bạn có thể được hỏi cài đặt thêm các gói.

Sau khi cài đặt xong, cửa sổ sau sẽ được hiển thị:


Cài đặt winestricks

Cài đặt Winetricks thông qua kho Ubuntu Software.

sudo apt install winetricks

Trong các lệnh ở trên mình cài luôn winetricks là một công cụ giúp chúng ta cài đặt các thư viện của Windows dễ hơn. Nó cung cấp cả giao diện đồ hoạ và thao tác trên terminal luôn. Đôi khi trong quá trình cài đặt ứng dụng nào đó nó sẽ báo lỗi thiếu thư viện dll hoặc .Net framework. Đó là lúc winetricks phát huy công dụng của nó.


Chạy ứng dụng Windows trên Linux với WINE

Chạy bằng lệnh:

wine windows_program.exe

Chúng ta cũng có thể nhấn chuột phải và chọn "Open with WINE Windows Program Loader".
