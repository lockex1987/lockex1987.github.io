Cài đặt Node

Cài đặt Node trên Ubuntu 20.10.

Các cách cài đặt

Cài đặt từ phiên bản distro-stable của Ubuntu

Phiên bản Node của Ubuntu 18.03 là 8.10.0. Đây không phải là phiên bản mới nhất.

sudo apt update
sudo apt install nodejs
sudo apt install npm

Chú ý: vì bị conflict với package khác, file chạy từ Ubuntu repo được đặt tên là nodejs chứ không phải node.

Kiểm tra:

nodejs -v

Phiên bản ở Ubuntu 18.04 chỉ là 8.10.0.

Cài đặt sử dụng một PPA

Để cài đặt phiên bản mới hơn của Node, bạn có thể thêm PPA (personal package archive) được maintain bởi NodeSource. Phiên bản này sẽ mới hơn phiên bản chính thức của Ubuntu repo, và cho phép bạn chọn các phiên bản Node (6.x, 8.x, 10.x,...).

cd ~
curl -sL https://deb.nodesource.com/setup_10.x -o nodesource_setup.sh
sudo bash nodesource_setup.sh
sudo apt install nodejs

Cài đặt sử dụng NVM

Một cách khác để thay cách cài với apt là sử dụng công cụ nvm (Node.js Verson Manager). Bạn có thể cài đặt nhiều phiên bản Node cùng một lúc và switch việc sử dụng giữa các phiên bản.

Cài đặt nvm:

curl -sL https://raw.githubusercontent.com/creationix/nvm/v0.33.11/install.sh -o install_nvm.sh
bash install_nvm.sh

# hoặc

wget -qO- https://raw.githubusercontent.com/creationix/nvm/v0.34.0/install.sh | bash

Áp dụng các thay đổi:

source ~/.profile

Cài từ file ZIP

Dùng cách này có thể cài offline, không cần mạng.

Download ở địa chỉ sau:

nodejs.org/en/download

Giải nén.

Thiết lập biến môi trường PATH ở file /etc/profile. Ví dụ:

PATH=$PATH:/home/lockex1987/program/node/bin
export PATH

Để có thể hiệu lực ngay, thực hiện lệnh:

source /etc/profile

Sử dụng nhiều phiên bản với NVM

Các phiên bản NVM

Tình hình là tôi đang sử dụng Node phiên bản 12. Tuy nhiên, tôi cần bảo trì một dự án sử dụng Node phiên bản 10. Giờ mà chỉnh lại các package phụ thuộc trong package.json rất khó mà cũng rất nguy hiểm.

Sử dụng nhiều phiên bản với NVM.

Có 2 repository NVM khác nhau, độc lập, nhưng các câu lệnh sử dụng giống nhau.

GitHub - nvm-sh/nvm: Node Version Manager - POSIX-compliant bash script to manage multiple active node.js versions

GitHub - coreybutler/nvm-windows: A node.js version management utility for Windows. Ironically written in Go.

Các câu lệnh NVM

Liệt kê các phiên bản có:

nvm ls-remote

Cài đặt một phiên bản nào đó:

nvm install <version>

Kiểm tra phiên bản hiện tại:

node -v
# hoặc
nvm current

Liệt kê các phiên bản đã cài đặt

nvm ls

Sử dụng phiên bản nào đó:

nvm use <version>

Gỡ bỏ phiên bản nào đó:

nvm deactivate
nvm uninstall <version>

Gỡ bỏ Node

Bạn có thể sử dụng apt hoặc nvm.

sudo apt remove nodejs
