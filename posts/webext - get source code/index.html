Lấy source code của một extension

Cách 1

Sử dụng extension:

Chrome extension source viewer - Chrome Web Store

Cách 2

Xem thư mục mà Chrome lưu ở ổ cứng trên máy.

Windows XP:

C:\Documents and Settings\%USERNAME%\Local Settings\Application Data\Google\Chrome\User Data\Default\Extensions\<Extension ID>

Windows 10/8/7/Vista:

C:\Users\%USERNAME%\AppData\Local\Google\Chrome\User Data\Default\Extensions\<Extension ID>

macOS:

~/Library/Application Support/Google/Chrome/Default/Extensions/<Extension ID>

Linux:

~/.config/google-chrome/Default/Extensions/<Extension ID>

Để xem ID của extension, bạn có thể vào địa chỉ:

chrome://extensions
