Font Awesome

Trang chủ

Font Awesome có license phức tạp. Để sử dụng đầy đủ icon bạn phải dùng bản Pro mất phí. Nếu chỉ dùng bản Free thì các icon ít hơn, chủ yếu là dạng solid nên trông cũng xấu.

Line Awesome thì miễn phí nhưng mà ít icon hơn, ít style hơn.

Nên dùng web font icon để căn chỉnh theo chiều dọc dễ hơn so với ảnh.

Sử dụng icon trên web

Icon đẹp, làm trang web màu sắc rực rỡ.

Dùng HTML entity thì có thể các font chữ khác nhau thể hiện khác nhau.

Dùng fonticon có thể đổi màu dễ dàng (color, background-color, căn dọc).

Không nên sử dụng icon ở trong button.

Icon processing sử dụng Bootstrap spinner. Demo.

Sử dụng hamburger bar riêng, có animation. Demo.
