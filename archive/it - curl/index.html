Câu lệnh curl

Demo

Method GET.

curl http://localhost/posts/php%20-%20mock%20api/get_demo.php?name=NAT

Method POST.

curl --request POST --data "name=NAT" http://localhost/posts/php%20-%20mock%20api/post_form_demo.php
curl -X POST --data "name=NAT" http://localhost/posts/php%20-%20mock%20api/post_form_demo.php
curl -X POST -d "name=NAT&age=23" http://localhost/posts/php%20-%20mock%20api/post_form_demo.php

Method DELETE.

curl -X DELETE http://localhost/posts/php%20-%20mock%20api/response_json_demo.php

Method PUT.

curl -X PUT http://localhost/posts/php%20-%20mock%20api/response_json_demo.php

Upload file.

curl --form uploadImage=@curl.png http://localhost/posts/php%20-%20mock%20api/post_upload_demo.php
curl --form uploadImage=@./curl.png http://localhost/posts/php%20-%20mock%20api/post_upload_demo.php

Lấy header thôi. 

curl -I http://localhost/posts/php%20-%20mock%20api/response_json_demo.php
