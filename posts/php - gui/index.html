Lập trình giao diện GUI với PHP

Tôi chuyên lập trình web. Tuy nhiên, có thể một lúc nào đó bạn sẽ cần làm một ứng dụng đơn giản, cần giao diện thay vì dòng lệnh. Bạn cũng có thể lập trình GUI với PHP.

Có nhiều thư viện:

PHP-GTK

wxPHP

cztomczak/phpdesktop: Develop desktop GUI applications using PHP, HTML5, JavaScript and SQLite

Tuy nhiên, đa phần các thư viện là cũ, chỉ hỗ trợ PHP bản cũ, hoặc cần cài đặt phức tạp.

Bạn nên sử dụng PHP-GUI:

PHP-GUI Documentation

gabrielrcouto/php-gui: Extensionless PHP Graphic User Interface library

Bạn chỉ cần dùng composer. Hỗ trợ cả Windows và Linux.

Không dùng GNOME hay Electron, vì tôi đang muốn dùng PHP.
