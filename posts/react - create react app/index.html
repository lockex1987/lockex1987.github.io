Create React App

Tạo mới

Tạo project mới:

npx create-react-app my-app
cd my-app
npm start

Hoặc:

create-react-app my-app

Thiết lập thư mục source

Sửa file jsconfig.json:

{
    "compilerOptions": {
        "baseUrl": "./src"
    }
}

Đổi cổng mặc định 300

Cách 1: Sửa file .env:

PORT=9999

Cách 2: Sửa file package.json

Môi trường Linux:

"start": "PORT=9999 react-scripts start"

# hoặc

"start": "export PORT=9999 react-scripts start"

Môi trường Windows:

"start": "set PORT=9999 && react-scripts start"
