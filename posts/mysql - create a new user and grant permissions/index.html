Tạo mới người dùng và gán quyền trong MySQL

Hãy bắt đầu bằng việc truy cập MySQL shell và tạo mới người dùng:

create user 'huyennv9'@'%' identified by 'abc123a@';

Gán quyền:

grant all privileges on sso.* to 'huyennv9'@'%';

Hiệu lực các quyền đã cấp:

flush privileges;

Cấu hình MySQL cho phép truy cập từ xa

Sửa file mysqld.cnf, sửa cấu hình bind-address từ mặc định 127.0.0.1 thành 0.0.0.0.

sudo vi /etc/mysql/mysql.conf.d/mysqld.cnf

Restart lại dịch vụ:

sudo systemctl restart mysql

Select các user hiện tại:

select * from mysql.user;

Cập nhật lại:

update mysql.user set Host = '%' where User = 'root';

flush privileges;
