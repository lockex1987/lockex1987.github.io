Cài đặt và ví dụ bắt đầu

Robot framework được viết bằng Python. Để chạy được Robot framework, bạn cần cài đặt Python và pip (pip là công cụ để quản lý các package của Python). Bạn có thể tìm hiểu cách cài trên mạng.

Bạn thực hiện hai lệnh sau để xác nhận việc cài đặt thành công:

$ python --version
Python 3.9.5

$ pip --version
pip 20.3.4 from /usr/lib/python3/dist-packages/pip (python 3.9)

Tiếp theo, chúng ta sẽ cài đặt Robot framework. Chú ý: chúng ta sẽ cài đặt phiên bản 3.2.2, không sử dụng phiên bản 4. IDE RED không hỗ trợ Robot framework phiên bản 4, báo lỗi Unknown keyword.

Tốt nhất bạn cũng nên sử dụng phiên bản Python 3.8. Tôi sử dụng Python 3.9 và RED báo lỗi Dialogs (non-accessible).

Cài đặt Robot framework bằng lệnh sau:

pip install robotframework==3.2.2

Xác nhận:

$ robot --version
Robot Framework 3.2.2 (Python 3.9.5 on linux)

Tạo file Hello_World.robot như sau:

*** Test Cases ***
My First Test
    Log    Hello World

Thực hiện lệnh:

robot Hello_World.robot

Kết quả thực hiện trên máy tôi là:

$ robot Hello_World.robot

IDE

Trên các bài hướng dẫn trên mạng thường sử dụng các IDE sau:

  • Cài đặt wxPython, robotframework-ride (khó cài, không thông dụng)
  • Sử dụng Eclipse RED (có highlight syntax, auto-complete, có document khi hover, không hỗ trợ Robot framework phiên bản 4)
  • Sử dụng Visual Studio Code và extension, để có documentation có thể giữ Ctrl và click vào keyword để đến code Python
  • PyCharm

IDE chỉ là công cụ hỗ trợ thôi, không phải là cái chính. Về sau giả sử có sử dụng CI/CD thì chúng ta vẫn phải sử dụng lệnh. Do đó tôi sẽ sử dụng Visual Studio Code và đánh lệnh.
