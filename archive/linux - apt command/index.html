Hướng dẫn sử dụng lệnh apt

Giới thiệu về lệnh apt

Ubuntu là 1 hệ điều hành thuộc nhánh Debian Linux. Nó sử dụng các dpkg package để đóng gói các chương trình, ứng dụng, dành để cài đặt vào hệ thống.

Lệnh apt (Advanced Package Tool) là một công cụ dòng lệnh được sử dụng để quản lý các dpkg package trên hệ thống của Ubuntu.

Update thông tin package từ server

Chúng ta sử dụng lệnh apt update để cập nhật các thông tin mới nhất của các package về máy.

sudo apt update

From the terminal image above, you should notice three different types of lines:

  • Hit: This signifies that there is no package change since the last check. Simply put, it means that there is no newer version of the package.
  • Get: This means there is a package update(new version) available and it will download the details for this update, but not the update itself.
  • Ign: This means that the package has been ignored. This happens either because of an error or because the package is recent and there is no need to check it for updates.

Hiển thị danh sách các package có thể update

Sử dụng lệnh apt list:

sudo apt list --upgradable

Nâng cấp package

Nâng cấp toàn bộ package:

sudo apt upgrade

Nâng cấp chỉ một package:

sudo apt install --only-upgrade <package_name>

Chúng ta có thể kết hợp cả hai lệnh apt update và upgrade để nâng cấp toàn bộ package trên hệ thống:

sudo apt update && sudo apt upgrade -y

Chú ý: Tham số -y để tự động đồng ý nâng cấp.

Cài đặt mới package

Để cài đặt mới một package chúng ta sử dụng lệnh apt install:

sudo apt install <package_name>

Mặc định lệnh trên sẽ cài đặt với version mới nhất, còn nếu muốn cài đặt một package với một version cụ thể thì chúng ta làm như sau:

sudo apt install <package_name>=<version_number>

Cài đăt nhiều package cho một lần:

sudo apt install <package_1> <package_2>

Gỡ bỏ package

Để gỡ bỏ một package trên Ubuntu, sử dụng lệnh apt remove:

sudo apt remove <package_name>

Với lệnh apt remove thì hệ thống chỉ gỡ bỏ binary package, chứ ko xóa bỏ các file cấu hình.

Vì vậy muốn gỡ bỏ hoàn toàn sạch sẽ một package, chúng ta sử dụng lệnh apt purge:

sudo apt purge <package_name>

Tìm kiếm package

Sử dụng lệnh apt search:

sudo apt search <package_name>

Hiển thị chi tiết thông tin một package

Sử dụng lệnh apt show:

sudo apt show <package_name>

Liệt kê danh sách các pakage đã cài đặt

Sử dụng lệnh apt list:

apt list --installed

Dọn dẹp các package của hệ thống

Sử dụng lệnh apt autoremove:

sudo apt autoremove
