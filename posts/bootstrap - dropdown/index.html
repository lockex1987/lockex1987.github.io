Bootstrap dropdown

Cơ bản

Một dropdown là một menu có thể ấn hiện mà cho phép người dùng chọn một giá trị từ danh sách được định nghĩa trước.

Class .dropdown đánh dấu, bao ngoài một dropdown.

Để mở dropdown, sử dụng một nút hoặc một link với class .dropdown-toggle và thuộc tính data-toggle="dropdown".

Thêm class .dropdown-menu vào phần tử <div> mà thực sự chứa menu. Sau đó thêm class .dropdown-item vào từng phần tử trong menu.

Bạn không phải khởi tạo dropdown một cách tường minh.

Các tùy chọn khác

Class .dropdown-divider được sử dụng để ngăn cách các phần tử với một đường kẻ ngang.

Class .dropdown-header được sử dụng để thêm header bên trong menu.

Highlight một phần tử với class .active (thêm background màu lam).

Để disable một phần tử, sử dụng class .disabled. Khi đó phần tử sẽ có màu xám và icon "no-parking-sign" khi hover.

Class .dropdown-item-text được sử dụng để thêm một phần tử văn bản thuần, hoặc một link với style mặc định.

Bạn cũng có thể tạo menu bên phải hoặc bên trái bằng cách thêm class .dropright hoặc .dropleft vào phần tử dropdown. Chú ý caret/arrow sẽ được thêm tự động.

Để căn phải menu, thêm class .dropdown-menu-right vào phần tử .dropdown-menu.

Nếu bạn muốn menu hiển thị lên đằng trên thay vì đằng dưới, thay đổi phần tử div với class .dropup.
