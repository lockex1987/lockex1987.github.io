Mock API

Giả lập dữ liệu từ server. Tạo các REST API đơn giản, để thực hành cho serie bài REST client.

Tạo các service nhỏ theo hướng micro-service.

Để expose REST API có thể áp dụng các web framework riêng của từng ngôn ngữ (Laravel, Spring, Django, Express) hoặc các micro-framework.

Client:

  • JS từ browser (AJAX)
  • PHP (curl, ixudra curl, Guzzle)
  • Lệnh curl
  • Java
  • Web Socket
