Hướng dẫn sử dụng Robot framework

Cài đặt

Cài đặt Python và pip.

Xác nhận:

C:\Users\VTCC->python --version
Python 3.9.2

C:\Users\VTCC->pip --version
pip 20.2.3 from c:\users\vtcc-\appdata\local\programs\python\python39\lib\site-packages\pip (python 3.9)

Cài đặt Robot framework:

pip install robotframework --proxy="http://192.168.103.26:80"

Xác nhận:

C:\Users\VTCC->robot --version
Robot Framework 4.0.3 (Python 3.9.2 on win32)

Đường dẫn là:

C:\Users\VTCC-\AppData\Local\Programs\Python\Python39\Scripts


Sử dụng Python 3.8, Robot framework 3.2.2 (không dùng Python 3.9 và Robot framework 4, IDE RED báo đỏ Unknown keyword).

Python 3.9 và Robot framework 3.2.2 vẫn không được. Bị lỗi Dialogs (non-accessible).

How do I install a different Python version using apt-get? - Ask Ubuntu

Managing Multiple Versions of Python on Ubuntu 18.04

Gỡ bản 4:

pip uninstall robotframework

Cài lại bản 3.2.2:

pip install robotframework==3.2.2 --proxy="http://192.168.103.26:80"

Xác nhận:

C:\Users\VTCC->robot --version
Robot Framework 3.2.2 (Python 3.8.1 on win32)

C:\Users\VTCC->rebot --version
Rebot 3.2.2 (Python 3.8.1 on win32)



Ví dụ bắt đầu

Tạo file HelloWorld.robot như sau:

*** Test Cases ***
MyFirstTest
    Log    Hello World

Thực hiện lệnh:

robot HelloWorld.robot

IDE

  • Cài đặt wxPython, robotframework-ride (không dùng do khó cài, không thông dụng)
  • Sử dụng Eclipse RED (có highlight syntax, auto-complete, có document khi hover)
  • Sử dụng Visual Studio Code và extension
