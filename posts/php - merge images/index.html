Ghép ảnh bằng PHP

Ghép nhiều ảnh nhỏ thành một ảnh lớn với PHP.

Nhiều trang ảnh truyện bị cắt thành từng ảnh nhỏ. Ghép lại để thành một trang hoàn chỉnh.

Một số hàm hỗ trợ

extension_loaded(): Kiểm tra xem extension đã được load hay chưa. Để thao tác với ảnh chúng ta cần load extension gd (Graphics Draw).

getimagesize(): Lấy chiều rộng, chiều cao, cũng như thông tin khác (type, bits, mime) của ảnh.

imagesx(): Lấy chiều rộng ảnh.

imagesy(): Lấy chiều cao ảnh.

imagecreatefrompng(), imagecreatefromjpeg(): Tạo các đối tượng ảnh từ đường dẫn (ảnh PNG, JPG).

imagepng(), imagejpeg(): Output ảnh ra file hoặc trả về cho trình duyệt (ảnh PNG, JPG).

imagecreatetruecolor(): Tạo một ảnh mới.

imagecopy(): Copy một phần của ảnh nguồn sang ảnh đích.

So sánh các hàm imagecopy

imagecopy          ( resource $dst_im , resource $src_im , int $dst_x , int $dst_y , int $src_x , int $src_y , int $src_w , int $src_h ): bool
imagecopymerge     ( resource $dst_im , resource $src_im , int $dst_x , int $dst_y , int $src_x , int $src_y , int $src_w , int $src_h , int $pct ): bool
imagecopymergegray ( resource $dst_im , resource $src_im , int $dst_x , int $dst_y , int $src_x , int $src_y , int $src_w , int $src_h , int $pct ): bool

imagecopyresampled ( resource $dst_im , resource $src_im , int $dst_x , int $dst_y , int $src_x , int $src_y , int $dst_w , int $dst_h , int $src_w , int $src_h ): bool
imagecopyresized   ( resource $dst_im , resource $src_im , int $dst_x , int $dst_y , int $src_x , int $src_y , int $dst_w , int $dst_h , int $src_w , int $src_h ): bool

The way these functions differ is in the last parameter: imagecopy() always overwrites all the pixels in the destination with those of the source, whereas imagecopymerge() merges the destination pixels with the source pixels by the amount specified in the extra parameter:

0 means "keep the source picture fully"

100 means "overwrite with the source picture fully",

50 means "mix the source and destination pixel colours equally"

The imagecopy() function is therefore equivalent to calling imagecopymerge() and passing in 100 as the last parameter.
