Phân trang trong Bootstrap

Để tạo phân trang, thêm một phần tử ul với class .pagination. Sau đó thêm class .page-item cho từng phần tử li và class .page-link cho các link ở bên trong li.

Thêm class .active cho .page-item của trang hiện tại.

Thêm class .disable cho .page-item mà không thể click được.
