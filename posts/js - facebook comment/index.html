Thêm Facebook comment cho website.

Danh sách link

Danh sách app, tạo appk, publish app:

developers.facebook.com/apps

Tài liệu:

developers.facebook.com/docs/plugins/comments

Quản lý comment:

developers.facebook.com/tools/comments

Đăng nhập qua Facebook:

developers.facebook.com/docs/facebook-login/web

Các câu hỏi thường gặp:

developers.facebook.com/docs/plugins/faqs

Các bước tích hợp

Thêm thẻ meta:

<meta property="fb:app_id" content="1762996047326114" />

Thêm thẻ div:

<div id="fb-root"></div>

Thêm thẻ script:

<script id="facebook-jssdk" src="https://connect.facebook.net/vi_VN/sdk.js#xfbml=1&version=v2.10&appId=1762996047326114"></script>

/* Facebook comment */
