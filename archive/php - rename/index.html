Rename

Chương trình PHP để đổi lại tên file hàng loạt.

Câu lệnh:

php Rename.php <command> <folder> [option]

Trong đó command có thể có các giá trị sau:

  • pr: Prefix
  • pt: Postfix
  • lc: Lower case
  • lt: Left trim
  • rt: Right trim
  • et: Change extension
  • rc: Remove special characters ("-", ": ", or any)
  • sf: Sort file
  • z: Compress file
  • x: Giải nén các file
  • mc: Manga chapter
