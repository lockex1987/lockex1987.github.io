Phần mềm VirtualBox

Dùng VirtualBox để tạo môi trường lab.

Cài đặt VirtualBox

Cài bằng lệnh apt-get.

sudo apt install virtualbox

Cài bản mới nhất từ file .deb trên trang chủ.

Một số lệnh quản lý

Liệt kê danh sách toàn bộ máy ảo:

VBoxManage list vms

Liệt kê danh sách máy ảo đang chạy:

VBoxManage list runningvms

Khởi động máy ảo:

VBoxManage startvm <vm_name>

Khởi động máy ảo ở chế độ headless:

VBoxManage startvm <vm_name> --type headless

Tắt máy ảo:

VBoxManage controlvm <mv_name> poweroff

Ngoài ra còn rất nhiều lệnh khác để quản lý máy ảo với VBoxManage, sử dụng VBoxManage với tham số –help để biết thêm chi tiết:

VBoxManage --help
