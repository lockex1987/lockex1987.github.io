Nhóm input

Bạn có thể mở rộng các form-control bằng cách thêm text hoặc các nút vào phía trước, phía sau hoặc cả hai phía của bất kỳ input dạng này nào.

The .input-group class is a container to enhance an input by adding an icon, text or a button in front (.input-group-prepend) or behind (.input-group-append) the input field as a "help text".

Use the .input-group-text class to style the specified help text.

Layout:

div.input-group
    div.input-group-prepend
        span.input-group-text
            <Text hoặc icon>
    input.form-control
    div.input-group-append
        span.input-group-text
            <Text hoặc icon>

Stacked form (form dọc)

Mình hay làm các ứng dụng quản lý, do đó việc nhập liệu form là một thao tác cơ bản hay gặp. Ngày xưa hay sử dụng table để thiết lập form. Cách này không phải là không tốt. Code rất đơn giản.

Chúng ta hãy tìm hiểu code form theo chuẩn Bootstrap là như thế nào.

Một số nguyên tắc:

  • Ngoài cùng là thẻ form (để bắt được sự kiện nhấn Enter).
  • Bọc label và các field ở trong thẻ <div> với class .form-group.
  • Thêm class .form-control vào các field dạng text như input[type=text], input[type=password], textarea, select.
  • Không thêm .form-control cho checkbox, radio, file. Chúng ta bao bởi div.form-check và label.form-check-label. Các phần tử có class là .form-check-input.

Có 3 dạng form:

  • Form dọc: đây là dạng mặc định, từng thành phần trên một dòng
  • Form cùng dòng: cả label và field cùng trên một dòng
  • Form ngang: từng cặp label và field đứng cạnh nhau trên một dòng

Layout:

form<.form-inline>
    div.form-group
        label
        (input|select|textarea).form-control
        input[type=file]
        p.help-block

    div.form-group.form-check
        div.(checkbox|radio)
            label.form-check-label
                input[type=(checkbox|radio)].form-check-input
                <Check text>
 
    button.btn

Inline form (form cùng dòng)

Thêm class .form-inline vào thẻ form. Không có .form-group.

Nếu màn hình quá bé (nhỏ hơn 576px) thì lại chuyển thành form dọc.

Cần thêm margin nữa.

Form ngang

Chúng ta thêm các điều chỉnh sau:

  • Thêm class .form-horizontal vào thẻ <form>
  • Thêm class .control-label vào các thẻ <label>
  • Thêm các class .col-*-* vào thẻ <label>
  • Bọc các field ở trong thẻ <div> có class .col-*-*

Chú ý kể cả checkbox, radio, button cũng cần được bọc trong thẻ .form-group.

form.form-horizontal
    div.form-group
        label.(control-label col-*-*)
        div.col-*-* [.col-*-offset-*]
            (input|select|textarea).form-control
            input[type=file]
            p.form-control-static
            div.(checkbox|radio)
                label
                    input[type=(checkbox|radio)]
                    <Check text>
            label.(checkbox-inline|radio-inline)
                input[type=(checkbox|radio)]
                <Check text>
            button

Trạng thái

Bootstrap cung cấp các style cho các validation như success, warning, error. Để sử dụng, thêm .has-success, .has-warning, .has-error vào thành phần cha (.form-group).

Để thêm icon:

  • Thêm .has-feedback vào thành phần cha (.form-group)
  • Thêm một thẻ <span> icon với class .form-control-feedback vào phía sau <input>

Layout:

form
    div.(form-group has-feedback (has-success|has-warning|has-error))
        input.form-control
        span.(glyphicon glyphicon-* form-control-feedback)

Kích thước

Thiết lập chiều cao của các field bằng cách thêm class kiểu .input-lg và .input-sm.

Thiết lập chiều rộng bằng các class grid kiểu .col-lg-* và .col-sm-*.

Có thể thiết lập nhanh cả label và field bằng cách thêm .form-group-* vào thành phần <div class="form-group">.

Tương tự có thể thiết lập nhanh bằng cách thêm .input-group-sm hoặc .input-group-lg.
