Để liệt kê các extension đã cài đặt, dùng lệnh sau:

php -m

Cài đặt từ repo

Dùng lệnh apt bình thường:

sudo apt install php7.2-mysql

Cài đặt từ source

phpize là chương trình dùng để comile một module từ source.

Cài đặt phpize:

sudo apt install php7.2-dev

Giả sử compile rar extension.

Download source từ địa chỉ: PECL :: Package :: rar.

Sau đó thực hiện các câu lệnh sau:

unzip php-rar-4.0.0.zip
cd php-rar-4.0.0
phpize
./configure
make
#make install

Sau khi biên dịch và cài đặt thành công, một file có tên rar.so sẽ được tạo ra và đặt vào thư mục dành cho các extension của phiên bản PHP tương ứng.

Kiểm tra thư mục extension:

php -i | grep extension_dir

Copy file:

sudo cp modules/rar.so /usr/lib/php/20170718/

Kiểm tra vị trí file php.ini:

php -i | grep 'Configuration File'

Sau đó, bạn cấu hình file php.ini để load module này.

sudo echo "extension=rar.so" >> /etc/php/7.2/cli/php.ini
