Cài đặt Redis

Ubuntu

Cài từ repo của từng loại hệ điều hành. Ví dụ:

sudo apt install redis-server

Lệnh trên cài luôn cả redis-cli.

Kiểm tra:

./redis-cli
...> set foo bar
OK
...> get foo
"bar"
...> keys *

Để có thể kết nối Redis từ máy khác, hãy sửa file /etc/redis/redis.conf, sửa cấu hình bind 127.0.0.1 ::0, sửa thành:

bind 127.0.0.1 <ip máy cài Redis>

Sau đó restart lại dịch vụ:

sudo systemctl restart redis

Kết nối đến máy cài Redis như sau:

redis-cli -h <ip máy cài Redis>

Windows

Redis không hỗ trợ chính thức Windows. Tuy nhiên, nhóm Microsoft Open Tech có phát triển và bảo trì phiên bản Windows ở địa chỉ sau:

github.com/MSOpenTech/redis/releases

Nếu không cài đặt được thì có thể sử dụng phiên bản thấp, ví dụ 2.8.2402.

CentOS

Cài đặt:

sudo yum install epel-release
sudo yum install redis

Sử dụng:

sudo systemctl status redis

sudo systemctl start redis

sudo systemctl enable redis

Kiểm tra:

redis-cli ping
