Web server đơn giản

Nhiều khi bạn cần một web server thực sự:

  • Gọi AJAX sẽ không thể thực hiện ở file:// do lỗi bảo mật
  • Mock dữ liệu trả về từ server
  • Chia sẻ file giữa các máy
  • ...

Có nhiều cách thực hiện (Python, PHP, Node,...).

Chúng ta sẽ tạo một web server đơn giản ở địa chỉ http://localhost:8080 từ thư mục nào đó.

Python

Python 2.

python -m SimpleHTTPServer 8080

Python 3.

python -m http.server 8080

PHP

Sử dụng PHP built-in web server.

php -S localhost:8080

Để truy cập được từ máy khác thì thay localhost bằng 0.0.0.0.

php -S 0.0.0.0:8080

Script:

start_server.sh

Node

Dùng package http-server.

Cài đặt:

npm install http-server -g

Sử dụng:

http-server
