Sử dụng SQLite với PHP

Enable extension

Bạn phải enable SQLite extension ở file php.ini.

extension=pdo_sqlite

extension=sqlite3

Cài đặt SQLite trên Windows

Trên hệ điều hành Linux, bạn đã có sẵn lệnh sqlite3.

Trên hệ điều hành Windows, bạn cần download các tool ở đây. Ví dụ bạn download file sqlite-tools-win32-x86-3330000.zip.

Giải nén file zip, bạn sẽ có 3 file thực thi sau:

  • sqldiff.exe
  • sqlite3.exe
  • sqlite3_analyzer.exe

Bạn mở màn hình cmd, di chuyển đến thư mục giải nén, sau đó thực hiện lệnh:

sqlite3.exe
