Highlight source code syntax

Cool! Bạn có thể viết các bài viết hướng dẫn với các đoạn code được highlight.

Bạn có thể sử dụng một trong 2 thư viện:

Highlight.js

CodeMirror

Sử dụng Highlight.js

Cách sử dụng thư viện Highlight.js đơn giản nhất đó là thêm các file CSS, JS và gọi phương thức hljs.initHighlightingOnLoad().

<link rel="stylesheet" href="/path/to/styles/default.css">
<script src="/path/to/highlight.min.js"></script>
<script>hljs.initHighlightingOnLoad();</script>

Các đoạn code bên trong <pre><code> sẽ được highlight.

Copy đoạn code có màu từ các trang web vào file DOC hoặc tạo file PDF bằng công cụ wk<HTML>toPDF.

Tham khảo:

highlightjs.org

Sử dụng CodeMirror

CodeMirror được sử dụng rộng rãi: Typora (tính năng code fence), BlueGriffin, CodePen, dev tool của Firefox, Chrome,...

CodeMirror là một editor. Tuy nhiên tôi chỉ muốn sử dụng tính năng highlight (bằng cách chạy runMode).

Cho vào docs.js.

Lấy CSS mà Typora sử dụng để highlight code ở thư mục C:\Program Files\Typora\resources\style.

CDN của các file JS, CSS là:

https://cdnjs.com/libraries/codemirror

Tham khảo:

https://codemirror.net/demo/
