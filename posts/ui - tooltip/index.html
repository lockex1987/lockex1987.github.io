Tooltip

Tính năng

  • Không cần khởi tạo bằng JS
  • Highlight
  • Transparent background
  • HTML
  • Hiệu ứng xuất hiện (fade, expand)
  • Move theo mouse

Cách sử dụng

Thêm class .tooltip-container vào phân tử.

Trong phần tử, thêm một thẻ span hoặc div con có class .tooltip.

Thêm các class .left, .right, .top, .bottom vào .tooltip-container để điều chỉnh vị trí của tooltip.
