Cài đặt Composer

Composer được viết bằng PHP mã đóng gói dạng PHAR, nên muốn dùng Composer phải chắc chắn máy đã cài đặt PHP. Từ terminal gõ lệnh sau kiểm tra xem máy có PHP hay chưa, nếu chưa có bạn phài cài PHP trước.

php --version

Trên Linux

Dùng các lệnh sau:

curl -sS https://getcomposer.org/installer \
    | sudo php -- --install-dir=/usr/local/bin \
    --filename=composer

sudo chown -R $USER $HOME/.composer

Trên Windows

Download file composer.phar ở địa chỉ sau:

Download Composer

Tạo file composer.bat bên cạnh file composer.phar với nội dung sau:

@php "%~dp0composer.phar" %*

Thiết lập biến môi trường PATH trỏ đến thư mục chứa 2 file trên.

Kiểm tra

composer --version
