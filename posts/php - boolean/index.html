Các giá trị, hàm boolean trong PHP

Các giá trị falsy

Những giá trị được coi là falsy là những giá trị khi được ép về kiểu boolean sẽ cho giá trị là false. Bao gồm:

  • Giá trị boolean false
  • Xâu rỗng ""
  • Xâu "0"
  • Số nguyên 0
  • Số thực 0.0
  • Mảng không có phần tử
  • null (gồm cả những biến không được gán giá trị)
  • SimpleXML object được tạo từ tag rỗng

Ngoài những giá trị là falsy kể trên, thì tất cả các giá trị khác đều được coi là true (truthy).

Hàm isset, empty, is_null

Ba hàm hữu ích isset, empty và is_null đều trả về một giá trị boolean.

isset trả về true nếu một biến được gán giá trị và không phải là null.

empty trả về true nếu biến là trống không (một xâu rỗng, false, mảng không có phần tử, null, xâu "0", số 0, một biến chưa được gán giá trị).

is_null trả về true nếu biến là null.

is_null đối diện với isset, ngoại trừ một sự khác biệt mà isset có thể được áp dụng cho các biến không xác định, nhưng is_null chỉ với các biến được khai báo.

Giá trị của biến boolean isset empty is_null
false false true true false
Xâu rỗng "" false true true false
Xâu "0" false true true false
Số nguyên 0 false true true false
Số thực 0.0 false true true false
Mảng rỗng [] false true true false
null false false true true
true true true false false
Không gán giá trị ERROR - false false true ERROR - true

Toán tử Elvis

Toán tử Elvis trả về giá trị toán hạng bên trái nếu toán hạng bên trái là truthy, ngược lại trả về toán hạng bên phải.

Ví dụ:

foo = bar_1 ?: bar_2;

Sẽ tương đương với:

foo = bar_1 ? bar_1 : bar_2;

Toán hạng bên trái bar_1 chỉ tính toán một lần.

Nó được gọi là toán hạng Elvis vì giống mái tóc của Elvis Presley.

Toán tử null-coalesing

Toán tử null-coalescing:

foo = bar_1 ?? bar_2;

Sẽ tương đương với:

foo = isset(bar_1) ? bar_1 : bar_2;
