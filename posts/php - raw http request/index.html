Tạo raw HTTP request

Một HTTP request là một kết nối socket bình thường.

Nội dung HTTP request khi upload file là:

POST /api/user HTTP/1.1
Host: sso-passport.cttd.tk
Connection: Close
Content-Type: multipart/form-data; boundary=$boundary
Content-Length: 196488

--$boundary
Content-Disposition: form-data; name="fullName"

Nguyễn Văn Huyên
--$boundary
Content-Disposition: form-data; name="email"

lockex1987@gmail.com
--$boundary
Content-Disposition: form-data; name="phone"

0386519125
--$boundary
Content-Disposition: form-data; name="avatar"; filename="nvh3.jpg"
Content-Type: image/jpeg

<Nội dung file>
--$boundary--
