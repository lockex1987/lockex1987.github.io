Lấy câu SQL

Trong Laravel, bạn thường sử dụng DB query builder hoặc Eloquent để truy vấn CSDL.

Vậy có cách nào để lấy ra câu lệnh SQL thuần (để select trực tiếp khi debug chẳng hạn)?

Ví dụ, bạn có đoạn code:

DB::table('users')->get();

hoặc:

User::get();

Bạn muốn lấy ra câu lệnh:

SELECT * FROM users

Bạn có thể sử dụng phương thức toSql() của đối tượng QueryBuilder. Phương thức này trả về một string. Sau đó, bạn có thể lấy string này bằng dd() hoặc Log::info().

Ví dụ:

$sql = DB::table('users')->toSql();
$sql = User::getQuery()->toSql();
dd($sql);
