Rename

Chương trình PHP để đổi lại tên file hàng loạt.

Câu lệnh

php Rename.php <command> <folder> [option]

Trong đó command có thể có các giá trị sau:

pr Prefix
pt Postfix
lc Lower case
lt Left trim
rt Right trim
et Change extension
rc Remove special characters (:, !, _, or any)
csc Check special characters (:, !)
sf Sort file
z Compress file
x Giải nén các file
mc Manga chapter

Phar file format

Đóng gói PHP project (gồm nhiều file) thành một file phar cho dễ triển khai, có thể chạy kiểu tiến trình.

Đầu tiên, bạn phải cấu hình phar.readonly thành 0 hoặc Off ở file php.ini.

Phải thêm hàm realpath thì hàm is_dir, is_file, file_exists mới chạy đúng khi đóng gói file phar. Nếu chúng ta để đường dẫn là tương đối, PHP sẽ hiểu là tương đối với stub của phar, không phải tương đối với thư mục đang chạy.

Tạo file phar:

php create-phar.php

Lệnh trên sẽ tạo file app.phar ở thư mục dist.
