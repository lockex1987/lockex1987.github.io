Carousel

Tính năng

  • Tự động khởi tạo, chỉ cần khai báo cấu trúc HTML, không cần khởi tạo bằng JS
  • Không bắt buộc thuộc tính id
  • Hỗ trợ touch - drag
  • Responsive bằng các biến CSS
  • Hiển thị multi item
  • Hiển thị indicator
  • Có hàm gotoItem, autoPlay
  • Thêm sự kiện sau khi chuyển item (dùng chỗ my calendar)
  • Khi trỏ chuột thì không autoPlay

Bootstrap carousel đơn giản nhưng hỗ trợ ít.

Chú ý: Phải tự style cho nat-carousel-indicators và nat-carousel-control.

Cấu trúc HTML

.nat-carousel
    .nat-carousel-wrapper
        .nat-carousel-inner
            .nat-carousel-item
                img

    .nat-carousel-indicators
        [data-item-to]

    .nat-carousel-control[data-direction=prev]
    .nat-carousel-control[data-direction=next]
