Cài đặt nemo trong Ubuntu

Ubuntu mặc định sử dụng Nautilus.

Nemo là file manager mặc định của Cinnamon desktop environment.

Một số tính năng:

  • Toolbar có thể cấu hình
  • Open as root
  • Open terminal
  • Hiển thị tiến trình copy ở dialog riêng
  • Ghi nhớ cách hiển thị (list, grid, compact)

Cài đặt nemo:

sudo apt install nemo

Mở nemo:

nemo

Thiết lập nemo là file manager mặc định:

sudo apt-get install dconf-tools

# xdg-mime default nautilus.desktop inode/directory application/x-gnome-saved-search

xdg-mime default nemo.desktop inode/directory application/x-gnome-saved-search

# gsettings set org.gnome.desktop.background show-desktop-icons true

gsettings set org.gnome.desktop.background show-desktop-icons false

gsettings set org.nemo.desktop show-desktop-icons true
