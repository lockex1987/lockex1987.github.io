Làm rối code JS

Minimize và làm rối code JavaScript, không cho client đọc hiểu. Minimize cũng khiến cho con người khó đọc hiểu code, vì các tên biến đã bị rút gọn lại.

Làm rối chưa chắc đã giảm dung lượng file, thậm chí có khi còn tăng dung lượng.

https://grantwinney.com/minification-vs-obfuscation/

Hai thư viện:

  • Uglify JS
  • JavaScript Obfuscator

Cài đặt và sử dụng:

npm install uglify-es -g
uglifyjs sample1.js -o sample1.min.js
uglifyjs sample1.js -c -m -o sample1.min.js

npm install javascript-obfuscator -g
javascript-obfuscator sample1.js --output sample1.obfuscated.js --compact true --self-defending false

Bạn cũng có thể sử dụng trong code:

var fs = require('fs');
var uglify = require('uglify-js');

var result = uglify.minify(["file-1.js"]);

fs.writeFile("output.min.js", result.code, function (err) {
    if (err) {
        console.log(err);
    } else {
        console.log("File was successfully saved.");
    }
});

Có thể kết hợp với Gulp.

Tùy nhu cầu xem có cần sử dụng hay không.
