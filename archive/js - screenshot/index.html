Chụp ảnh màn hình

Sử dụng thư viện dom-to-image, html2canvas, rasterizeHTML hoặc API của web extension.

Thư viện html2canvas không hỗ trợ box-shadow, ảnh ở domain khác (phải thêm thư viện), tốc độ có vẻ chậm hơn thư viện dom-to-image.

Khi dùng thư viện html2canvas, có thể thêm thuộc tính data-html2canvas-ignore với các phần tử mà bạn không muốn được bao gồm ở trong screenshot.
