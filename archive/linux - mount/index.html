Mount ổ cứng

Chuẩn bị

Liệt kê danh sách ổ cứng:

lsblk

Lấy UUID của ổ cứng:

lsblk -o NAME,UUID,SIZE

Tạo thư mục để mount:

sudo mkdir /media/lockex1987/data

Thực hiện

Liệt kê danh sách đang được mount:

mount

Bỏ không mount nữa, dùng lệnh umount (u chứ không phải un):

sudo umount <FOLDER>

sudo umount /media/lockex1987/data

Mount ổ cứng:

sudo mount -t <TYPE> -o <OPTIONS> <DEVICE> <FOLDER>

sudo mount -t ext4 -o defaults /dev/sdb1 /media/lockex1987/data

-o defaults ở đây bao gồm rw, suid, dev, exec, auto, nouser, và async.

Bình thường là:

(rw,nosuid,nodev,relatime,data=ordered,uhelper=udisks2)

Khi chạy là:

(rw,relatime,data=ordered)

Cấu hình fstab

Cấu hình /etc/fstab để tự động mount ổ cứng khi khởi động máy.

# <file system>                              <mount point>             <type>  <options>            <dump>  <pass>
UUID=39e0205d-c528-4c82-8336-6296e76de697    /media/lockex1987/data    ext4    defaults             0       1
