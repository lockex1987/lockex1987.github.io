Cài đặt Laravel

Yêu cầu server

Để sử dụng Laravel framework thì server của bạn cần một số yêu cầu:

  • PHP >= 7.1.3
  • BCMath extension
  • Ctype extension
  • JSON extension
  • Mbstring extension
  • OpenSSL extension
  • PDO extension
  • Tokenizer extension
  • XML extension

Để kiểm tra xem các extension đã đầy đủ chưa, bạn sử dụng lệnh sau:

php -m

Với kết quả như trên thì các extension đã đầy đủ.

Bạn cũng có thể sử dụng script PHP sau:

check.php

Nếu chưa đầy đủ, chúng ta có thể cài đặt như sau:

sudo apt install php7.2-mbstring php7.2-xml \
    php7.2-json php7.2-zip php7.2-bcmath

Cài đặt Laravel

Laravel sử dụng Composer để quản lý các dependency. Vì vậy, trước khi bắt tay vào sử dụng Laravel thì máy tính của bạn phải cài Composer trước.

Bạn có thể sử dụng Composer create-project trên terminal:

composer create-project --prefer-dist laravel/laravel blog

Phát triển trên local

Nếu máy tính của bạn đã cài PHP rồi thì bạn có thể server PHP built-in để phát triển server ứng dụng của bạn bằng cách sử dụng lệnh artisan serve. Nó sẽ tạo ra một server tại địa chỉ localhost:8000:

php artisan serve
