Cấu hình proxy cho npm

Sử dụng các lệnh sau:

npm config list

npm config set proxy http://192.168.103.31:80
npm config set https-proxy http://192.168.103.31:80

npm config get proxy
npm config get https-proxy

npm config delete proxy
npm config delete https-proxy

npm install --proxy http://192.168.103.31:80 electron
