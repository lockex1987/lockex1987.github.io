User và group trong Linux

Lấy user hiện tại:

whoami

User hiện tại thuộc vào nhóm nào:

groups

Xem một user nào đó thuộc vào nhóm nào (thực hiện khi đang là root):

groups <username>

Gán user cho group:

usermod -g <groupname> <username>

Xóa user khỏi group:

gpasswd -d user group
deluser <username> <groupname>

Liệt kê tất cả user:

cut -d: -f1 /etc/passwd | sort

Liệt kê tất cả group:

cut -d: -f1 /etc/group | sort

Tạo user:

useradd -c "Tran Huy Hoang - admin2" tranhoang

Xóa người dùng:

userdel -r <username>

Tạo nhóm:

groupadd <groupname>

Xóa nhóm:

groupdel <groupname>
