PDF & DOC

Các định dạng

Có nhiều định dạng khác nhau:

  • text: XML, HTML, JSON, TXT, Markdown, RTF
  • binary: DOC, XLS, PPT, PDF

Mỗi định dạng lại đều có điểm mạnh, đặc trưng riêng.

Các định dạng text thì dễ chỉnh sửa.

Các định dạng binary thì để tiết kiệm dung lượng truyền, hay dùng làm đầu ra hiển thị.

Làm thế nào để qua lại giữa các định dạng với nhau (convert)? Làm thế nào để xuất báo cáo ra các định dạng đó (export)?

Thư viện

Một số thư viện hay được sử dụng là:

JOD Converter Convert qua lại giữa các định dạng.
Việc convert được dựa trên Open Office, tương tự như chúng ta mở file ra và Save as thành một định dạng khác.
Chất lượng tốt.
Flying Saucer Convert HTML sang PDF.
Chúng ta có thể thêm các tính năng như trang bìa, header, footer, TOC dễ dàng bằng cách sử dụng CSS Paged Media.

https://tuhrig.de/generating-pdfs-with-java-flying-saucer-and-thymeleaf/

iText Export, chỉnh sửa file PDF.
jSoup Parse, chỉnh sửa file HTML.
Chúng ta có thể chuẩn hóa code HTML thành XHTML, loại bỏ các tag nguy hiểm để tránh lỗi ATTT.
docx4j Export, chỉnh sửa file DOCX.
cttd-doc Export ra file DOCX qua file template.
