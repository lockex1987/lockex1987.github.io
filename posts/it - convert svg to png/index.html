Convert ảnh SVG thành PNG (resize luôn)

Nhiều khi tôi có nhu cầu convert ảnh SVG thành PNG với kích thước lớn:

  • Icon của web extension
  • Icon của Electron app trên Linux
  • Icon của app khi cài trên Linux

Khi đó, chúng ta có thể sử dụng Node package svgexport.

Cài đặt:

npm install svgexport -g

Sử dụng:

svg <input file> <output file> <options>

Ví dụ muốn convert ảnh SVG thành PNG với kích thước 256x256:

svg export input.svg output.png 256:256

Tham khảo:

shakiba/svgexport: SVG to PNG/JPEG command-line tool and Node.js module
