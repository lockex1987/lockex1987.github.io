Đổi style cho trường mà trình duyệt tự điền dữ liệu

Các trình duyệt cho phép người dùng tự động điền dữ liệu vào các trường của form, ví dụ như thông tin người dùng và mật khẩu khi đăng nhập. Ví dụ:

Chú ý các trường mà được tự điền dữ liệu có background màu vàng. Chúng ta sẽ thay đổi style này. Chúng ta sử dụng -webkit-autofill pseudo-selector như sau:

/* Change auto-complete styles in Chrome */
:-webkit-autofill,
:-webkit-autofill:hover, 
:-webkit-autofill:focus {
    /* Có thể thiết lập cả border */
    border: 1px solid green;

    /* Màu của chữ cũng bị thay đổi, chuyển về màu khi bình thường */
    -webkit-text-fill-color: #9ECCF8;

    /* Thời gian đổi màu trong 5000s (một thời gian dài) */
    transition: background-color 5000s ease-in-out 0s;

    /* Màu nền cùng màu nền của cha (nhưng sẽ bị nháy nháy) */
    /*-webkit-box-shadow: 0 0 0px 1000px #002864 inset;*/

    /* Dùng thế này không ăn */
    /*background-color: transparent !important;*/
}

Chúng ta có thể sử dụng box-shadow, hoặc để thời gian transition là một thời gian rất dài (làm cách này).

Một cách khác là sử dụng keyframes:

@-webkit-keyframes autofill {
    to {
        color: #666;
        background: transparent;
    }
}

input:-webkit-autofill {
    -webkit-animation-name: autofill;
    -webkit-animation-fill-mode: both;
}
