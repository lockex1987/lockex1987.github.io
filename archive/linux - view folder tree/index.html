Hiển thị cây thư mục

Lệnh ls chỉ hiển thị dữ liệu của một cấp. Để hiển thị dữ liệu tất cả các cấp dạng cây, sử dụng lệnh tree.

Lệnh tree không được cài đặt mặc định. Bạn có thể cài như sau:

sudo apt install tree

Cấu trúc câu lệnh như sau:

tree /path/to/folder

Đầu ra dạng như sau:
