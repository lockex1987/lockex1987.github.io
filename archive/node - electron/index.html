Electron

Mục tiêu: Viết ứng dụng xem ảnh, cbr, cbz như Comix. Cross platform. Dùng Node viết bằng JS cho sướng. Không học thêm Python.

Tính năng

Tên eComix (e là Electron).

Phím tắt

  • A: Actual size
  • B: Best fit
  • W: Fit width
  • H: Fit height?
  • H: Home (hiển thị danh sách ảnh dạng grid)
  • F: Fullscreen toggle
  • N: toggle filename
  • ESC: exit fullscreen hoặc thoát chương trình
  • -> Page Down: Next page
  • <- Page Up: Previous page

Cài đặt

Nếu máy bạn kết nối mạng qua proxy thì phải thiết lập như sau:

set GLOBAL_AGENT_HTTP_PROXY=http://192.168.103.33:80/
set GLOBAL_AGENT_HTTPS_PROXY=http://192.168.103.33:80/
set ELECTRON_GET_USE_PROXY=true

Câu lệnh cài đặt:

npm i --save-dev electron
