Ẩn mũi tên ở thẻ input number

Thẻ input có thể có type là "number". Khi thiết lập type="number", chúng ta chỉ có thể nhập số hoặc dấu chấm, đỡ một số validate. Ở màn hình điện thoại cũng chỉ hiển thị các số cho chúng ta nhập.

Ví dụ:

Chúng ta có thể tăng giảm giá trị bằng cách click vào phím mũi tên.

Đôi lúc, việc hiển thị mũi tên là không mong muốn, nó làm thay đổi so với design.

Chúng ta có thể thêm CSS như sau:

input[type=number].no-arrows::-webkit-inner-spin-button, 
input[type=number].no-arrows::-webkit-outer-spin-button { 
    -webkit-appearance: none;
    margin: 0;
}

input[type="number"].no-arrows {
    -moz-appearance: textfield;
}

Ví dụ:

Chúng ta vẫn có thể tăng giảm giá trị bằng cách nhấn các nút mũi tên trên bàn phím.
