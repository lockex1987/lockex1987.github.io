Mở web ở chế độ app

Bạn có thể mở một trang web ở chế độ app với Chrome (Chromium), tạo cảm giác như đang sử dụng một ứng dụng desktop. Nó sẽ tạo một cửa sổ mà không có toolbar, không có thanh địa chỉ.

Windows:

Chrome.exe --app=https://google.com

Mac:

/Applications/Google\
        Chrome.app/Contents/MacOS/Google\ Chrome --app=https://google.com

Linux:

google-chrome --app=https://google.com

Bạn cũng có thể mở từ một extension:

chrome.windows.create({
    url: '<the url>',
    type: 'popup',
    width: 800,
    height: 600
});

Tạo desktop shortcut, menu item:

google-chrome --app=http://sso-admin.cttd.tk/
        --window-size=1500,800 --window-position=200,50
