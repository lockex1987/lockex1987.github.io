Electron

Cài đặt

Nếu máy bạn kết nối mạng qua proxy thì phải thiết lập như sau:

set GLOBAL_AGENT_HTTP_PROXY=http://192.168.103.31:80/
set GLOBAL_AGENT_HTTPS_PROXY=http://192.168.103.31:80/
set ELECTRON_GET_USE_PROXY=true

Câu lệnh cài đặt:

npm i --save-dev electron
